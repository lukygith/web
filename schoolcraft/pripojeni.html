Jak se připojit na server

Server **SchoolCraft.cz** je určen pro studenty naší školy. Abyste se mohli připojit, musíte splnit následující podmínky:

  • Být studentem naší školy: Server je přístupný pouze studentům naší školy. Pokud nemáte školní e-mail, připojení nebude možné.
  • Verze Minecraftu: Pro připojení musíte mít nainstalovanou verzi Minecraftu kompatibilní se serverem (nejnovější verze).
  • IP adresa serveru: Server je k dispozici na adrese northern-sept.gl.joinmc.link.
  • Pokud nemáte účet: Pokud nemáte účet, zaregistrujte se přes školní systém a požádejte administrátory o přístup.

Pokud máte problémy s připojením

Pokud máte jakékoliv problémy s připojením, obraťte se na školní administrátory nebo nás kontaktujte přes náš Discord server pro podporu.
